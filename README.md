# ESP32 Blink WiFi OTA Monitor

Projeto PlatformIO para ESP32 com:

- OTA via WiFi
- Blink sem delay
- Infraestrutura para monitoramento de câmeras IP
- Arquitetura preparada para IoT supervision

## Hardware
- ESP32 DevKit
- WiFi 2.4GHz

## Features
- ArduinoOTA
- FreeRTOS Ready
- Non-blocking loop
- Expansível para MQTT e RTSP watchdog
